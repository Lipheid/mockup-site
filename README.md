# 콘텐츠ON 정적 목업

기업의 광고, 홍보물, 영상, SNS 게시물 등 디지털 콘텐츠에 연결된 민원을 AI로 분석하고 처리하는 프레임워크 없는 웹사이트 목업입니다.

## 실행

`index.html`을 브라우저에서 열면 됩니다. 별도의 설치나 빌드 과정이 없습니다.

관리자 로그인 화면에는 목업용 계정이 미리 입력되어 있어 **로그인** 버튼만 누르면 관리자 콘솔로 전환됩니다.

## 구현 화면

- 민원인: 메인, 서비스 소개, 콘텐츠 민원 접수, 처리 현황 조회, 이용 안내
- 관리자: 로그인, 대시보드, 콘텐츠 민원 목록/상세, AI 정제 및 분석, 콘텐츠 이슈 클러스터, 위험 민원, 공식 안내문 작성, 시스템 설정
- 반응형 레이아웃 및 기본 클릭 인터랙션
